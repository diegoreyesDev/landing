Acerca de mi

Soy Diego Reyes, desarrollador Full Stack Java con experiencia en aplicaciones web dinamicas, JEE, Spring Boot, Spring MVC, Spring Security y JPA, asi como en bases de datos Oracle. Tambien tengo conocimientos en HTML, CSS y JavaScript.

Complemento mi perfil tecnico con experiencia en gestion cultural, gestion de proyectos e instruccion en artes marciales, lo que me aporta una vision integral para el trabajo en equipo y la resolucion de problemas complejos.

Actualmente vivo en Talagante, Region Metropolitana (Chile), donde me desempeno como gestor cultural enfocado en la digitalizacion de datos para organizaciones comunitarias. Me apasiona el desarrollo de aplicaciones Java y busco perfeccionarme continuamente para crear soluciones que reduzcan la friccion del usuario final.
